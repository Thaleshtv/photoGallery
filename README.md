# MyGallery

O projeto trata-se da criação de um albúm de fotos responsivo que contém a principio fotos aleatórias fornecidas pela API e também a opção de pesquisa para as fotos desejadas pelo usuário. Foi desenvolvido em Reactjs utilizando para estilização o Styled components.

# Estruturação
Foi divido em 4 pastas (components, pages, services e styles). Na pasta services se encontram as chamadas para API, separadamente, para fotos aleatórias e buscadas pelo usuário.
