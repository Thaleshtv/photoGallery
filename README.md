# MyGallery

O projeto trata-se da criação de um álbum de fotos responsivo que contém a principio fotos aleatórias fornecidas pela API e também a opção de pesquisa para as fotos desejadas pelo usuário. Foi desenvolvido em Reactjs utilizando para estilização o Styled components.

# Estruturação
Foi divido em 4 pastas (components, pages, services e styles). 
OBS: Na pasta services encontram-se as chamadas para API, separadamente, para fotos aleatórias e buscadas pelo usuário.
